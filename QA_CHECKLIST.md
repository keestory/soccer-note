# SoccerNote - QA 체크리스트

## 1. 인증 기능 테스트

### 회원가입
- [ ] 유효한 이메일과 비밀번호로 회원가입 성공
- [ ] 6자 미만 비밀번호로 가입 시 에러 메시지 표시
- [ ] 비밀번호 확인 불일치 시 에러 메시지 표시
- [ ] 이미 사용 중인 이메일로 가입 시 에러 처리
- [ ] 가입 성공 후 로그인 페이지로 리다이렉트

### 로그인
- [ ] 올바른 자격 증명으로 로그인 성공
- [ ] 잘못된 비밀번호로 로그인 시 에러 메시지 표시
- [ ] 존재하지 않는 이메일로 로그인 시 에러 처리
- [ ] 로그인 성공 후 대시보드로 리다이렉트

### 로그아웃
- [ ] 로그아웃 버튼 클릭 시 세션 종료
- [ ] 로그아웃 후 랜딩 페이지로 리다이렉트
- [ ] 로그아웃 후 보호된 페이지 접근 불가

---

## 2. 팀 관리 테스트

### 팀 생성
- [ ] 첫 로그인 시 팀 생성 화면 표시
- [ ] 팀 이름 입력 후 생성 성공
- [ ] 빈 이름으로 생성 시 유효성 검사

---

## 3. 선수 관리 테스트

### 선수 추가
- [ ] 이름, 등번호, 포지션 입력 후 추가 성공
- [ ] 이름 없이 추가 시 유효성 검사
- [ ] 등번호는 선택 사항
- [ ] 포지션 선택 (GK, DF, MF, FW)

### 선수 수정
- [ ] 기존 선수 정보 편집 가능
- [ ] 수정 사항 저장 성공

### 선수 삭제
- [ ] 삭제 확인 다이얼로그 표시
- [ ] 삭제 성공 후 목록에서 제거

---

## 4. 경기 관리 테스트

### 경기 생성
- [ ] 상대팀, 날짜, 장소 입력 후 생성 성공
- [ ] 상대팀 이름 필수 입력 검증
- [ ] 생성 시 4쿼터 자동 생성 확인
- [ ] 생성 후 경기 상세 페이지로 이동

### 경기 상세 보기
- [ ] 스코어 표시
- [ ] 쿼터별 탭 전환
- [ ] MVP 표시 (기록이 있을 경우)
- [ ] 선수별 통계 표시

### 경기 삭제
- [ ] 삭제 확인 다이얼로그 표시
- [ ] 삭제 시 관련 쿼터/기록 함께 삭제

---

## 5. 쿼터 편집 테스트

### 포메이션 배치
- [ ] 선수 추가 버튼 클릭 시 선수 목록 표시
- [ ] 선수 선택 시 필드에 추가
- [ ] 드래그로 선수 위치 이동
- [ ] 터치 디바이스에서 드래그 동작
- [ ] 같은 선수 중복 추가 방지

### 선수 평가
- [ ] 선수 클릭 시 평가 패널 표시
- [ ] 평점 슬라이더 0.0~10.0 조절
- [ ] 골, 어시스트 숫자 입력
- [ ] 클린시트 토글
- [ ] 기여도 입력

### 저장
- [ ] 저장 버튼 클릭 시 데이터 저장
- [ ] 저장 성공 메시지 표시
- [ ] 저장 후 경기 상세로 이동

---

## 6. MVP 계산 테스트

- [ ] 모든 쿼터의 평균 점수 계산
- [ ] 가장 높은 평균 점수 선수를 MVP로 표시
- [ ] 점수가 없는 선수는 MVP 후보에서 제외
- [ ] MVP 뱃지 및 점수 표시

---

## 7. UI/UX 테스트

### 반응형 디자인
- [ ] 모바일 (< 640px) 레이아웃 확인
- [ ] 태블릿 (768px~1024px) 레이아웃 확인
- [ ] 데스크탑 (> 1024px) 레이아웃 확인

### 축구장 UI
- [ ] 축구장 비율 3:2 유지
- [ ] 필드 라인 표시
- [ ] 선수 마커 포지션별 색상
- [ ] 선수 마커 드래그 시 시각적 피드백

### 일반 UI
- [ ] 버튼 hover 상태
- [ ] 로딩 스피너 표시
- [ ] 토스트 알림 표시
- [ ] 폼 입력 필드 포커스 상태

---

## 8. 에러 핸들링 테스트

- [ ] 네트워크 오류 시 에러 메시지 표시
- [ ] 잘못된 URL 접근 시 리다이렉트
- [ ] 권한 없는 데이터 접근 시 처리
- [ ] 입력 유효성 검사 에러 표시

---

## 9. 보안 테스트

- [ ] 비로그인 상태에서 보호된 페이지 접근 차단
- [ ] 다른 사용자의 데이터 접근 불가 (RLS 확인)
- [ ] API 키가 클라이언트에 노출되지 않음

---

## 10. 성능 테스트

- [ ] 페이지 로드 시간 < 3초
- [ ] 대량 선수 (20명+) 목록 스크롤 원활
- [ ] 드래그 앤 드롭 반응 지연 없음
